# Tai Huynh — Portfolio Site

Personal portfolio for Tai Huynh, an AI & full-stack engineer building production-grade AI features.

## Stack

- Next.js 14 (App Router, `src/` directory)
- TypeScript (strict)
- Tailwind CSS 3
- Static export (`next build` → `/out`)

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Output is in `/out` — deploy to Vercel, Cloudflare Pages, or any static host.

## Deploy to Vercel

Push to GitHub and import the repo in Vercel. No environment variables needed.
